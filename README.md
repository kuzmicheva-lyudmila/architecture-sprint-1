# Проект "MESTRO"

## Функциональность:
- загрузка фотографий
- удаление фотографий
- лайки под фото
- работа с профилем пользователя

## Разделение на микрофронтенды
Для разделения на микрофронтеды используется ветикальная нарезка: разделение на домены.
Выделятся следующие домены:
- Домен аутентификации: регистрация/подключение пользователя
- Домен работы с фотографией: загрузка, удаление, лайки
- Домен работы с профилем: редактирование профиля, изменение аватарки

Для работы с микрофронтедами используется **Webpack Module Federation**. Т.к. проект небольшой и на текущем этапе его
развития нет смысла использовать разные фреймворки.
Также есть общие части: PopupWithForm (всплывающая форма), стили css, набор шрифтов

Для связи между микрофронтендами используется библиотека глобального состояния Redux. В Redux Store помещается информация о текущем пользователе CurrentUserContext
